Solução automatizada para gerenciamento e comunicação das agendas de atualização dos ambientes municipais, integrada ao Google Sheets.

Desenvolvida para garantir maior controle e visibilidade sobre as atualizações programadas, a ferramenta organiza e distribui as informações de forma padronizada e acessível à equipe.

Principais funcionalidades:
Integração com planilhas Google para leitura automatizada da agenda
Consolidação e organização das atualizações por horário
Envio de notificações estruturadas via Telegram contendo:
Data da agenda
Versão homologada
Municípios programados para atualização
Identificação do chamado relacionado
Versão prevista para implantação
Diferenciais operacionais:
Agrupamento por janelas de execução (ex: 11:00, 17:00, 19:00)
Padronização das comunicações para facilitar interpretação
Destaque de informações críticas ou específicas

A solução otimiza o acompanhamento das atualizações, reduz riscos operacionais e melhora a sincronização entre as equipes envolvidas no processo.
